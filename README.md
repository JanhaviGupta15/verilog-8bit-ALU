# 4-bit ALU using Verilog

This project implements a 4-bit Arithmetic Logic Unit using Verilog HDL.

Operations implemented:
- Addition
- Subtraction
- Multiplication
- Logical operations (AND, OR, XOR, NAND, NOR)
- Bit shifts
- Comparison
- Division (using a custom divider module)

Tools Used:
- Verilog HDL
- ModelSim
- Xilinx ISE
